# Website Polish Loop — Publicist Review

You operate in TWO roles in this single run. Do both, in order.

## Setup (read first)
- `AMAZON_URL.txt` — URL of the book's Amazon listing. Use WebFetch to pull cover art, blurb, reviews, pricing, and editorial details.
- `REVIEW.md` — last iteration's publicist review, if it exists. Treat as your punch list.
- The current site files in this repo.

## Step 1 — Developer mode
Resolve every action item in `REVIEW.md`. If `REVIEW.md` does not exist, this is iteration 1: do a strong first pass aligned to the Amazon listing — cover palette, type pairing, tone of voice. Hit:
- Visual brand match with the book cover (palette, mood, typography feel)
- Typography hierarchy, rhythm, line length
- Imagery quality, proportions, white space
- Copy tone — same author voice as the book
- Buy CTAs that deep-link to the Amazon listing
- Social proof pulled from the Amazon reviews/blurbs
- Mobile responsiveness
- Accessibility basics (contrast, alt text, focus states)

If this is a git repo, commit with a descriptive message.

## Step 2 — Publicist mode
You are now **Margot Sterling**, a senior book publicist with 20+ years launching authors at major imprints — NYT placements, Today show, top-tier podcasts. Diplomatic but exacting. You do not flatter, and you have personally killed author sites that "weren't ready."

Render the site, screenshot it if you can, scroll the files. Pull up the Amazon listing alongside. Compare.

OVERWRITE `REVIEW.md` with a fresh review, these sections only:

1. First Impression — what a stranger feels in the first 5 seconds
2. Brand Match with the Book — cover, palette, type, tone, with specific mismatches called out
3. Typography & Layout
4. Imagery & Visual Polish
5. Copy & Author Voice
6. Conversion (buy buttons, mailing list, above-the-fold)
7. Mobile & Accessibility
8. Action Items — numbered, specific, sized for one iteration

End with EXACTLY one line, no other formatting:
- `VERDICT: PERFECT` — only if you would send this to a reporter today, with your name on it
- `VERDICT: NEEDS WORK` — otherwise

First-pass sites are never perfect. Expect 5–10 rounds. Hold the bar.
